# Day 10 - for 循环

## 学到的概念

### 1.`for` 循环的基本结构

```python
for 变量 in 可迭代对象:
    循环体
```

- Python 会依次取出可迭代对象中的每一个元素，赋值给元素变量，然后执行循环体
- 可迭代对象：字符串、列表、元组、字典、range()等

### 2.`range()` —— 生成整数序列

- `range()` 可以用来生成一串数字，非常适合做固定次数的循环
- **语法**：`range(start, end, step)`
    - start：省略默认为 `0`
    - stop：省略默认为 `end - 1`
    - step：省略默认为 `1`

### 3.嵌套循环

一个循环里面再放一个循环，可以处理二维模式（如九九乘法表）

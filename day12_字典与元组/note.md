# Day 12 - 字典与元组

## 学到的概念

### 1.字典（Dict）—— 键值对的集合

- 列表用数字索引来查找元素，而字典用**自定义的键**来查找值
- 就像现实中的字典：根据“词语”查找对应的“释义”
- **创建字典**：

```python
person = {
    "name": "小明",
    "age": 25,
    "city": "广州"
}
print(person["name"])  # 小明
print(person["age"])  # 25
```

- **键（key）**：不可变类型
- **值（value）**：可以是任意类型
- 键必须唯一，值可以重复

### 2.字典的常用操作

- `dict[key] = value`：键存在就是修改；不存在就是添加
- `del dict[key]`：删除指定键值对
- `dict.pop(key, None)`：安全删除，键不存在不报错
- `dict.get(key, 默认值)`：查找键对应的值，如果不存在返回默认值
- `dict.keys()`：获取所有键
- `dict.values()`：获取所有值
- `dict.items()`：获取所有键值对

### 3.元组（Tuple）—— 不可变的列表

元组和列表几乎一样，但**一旦创建就不能修改**

#### 为什么需要不可变的数据?

- 保护数据不被意外修改
- 可以作为字典的键（列表不行）
- 当数据固定不变时，用元组更安全、更省内存

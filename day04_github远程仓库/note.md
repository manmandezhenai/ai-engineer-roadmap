# Day 4 - GitHub 远程仓库

## 学到的概念

- GitHub 是代码托管平台，可以备份、记录历史、协作
- 本地仓库 vs 远程仓库：push 上传，pull 下拉
- origin 是远程仓库的默认名称
- 首次推送需要 -u 建立跟踪关系

## 学会的命令

- github.com 创建远程仓库
- `git remote add origin <URL>`：关联远程仓库
- `git remote -v`：查看远程仓库地址
- `git branch -M main`：将分支改名为 main
- `git push -u origin main`：首次推送并建立跟踪
- `git push`：之后的日常推送

## 日常提交流程

- `git add .`
- `git commit -m "描述"`
- `git push`

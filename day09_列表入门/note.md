# Day 9 - 列表入门

## 学到的概念

### 1.什么是列表?

列表（List）是 Python 中一种**有序、可变、可重复**的容器数据类型，用来存放一组数据。

### 2.创建列表

在 Python 中，列表用方括号 `[]` 表示，元素之间用逗号分隔

```python
fruits = ["苹果", "香蕉", "橘子"]
numbers = [10, 20, 30, 40]
mixed = [1, "hello", 3.14, True]  # 可以混装类型（但不推荐）
empty = []  # 空列表
```

### 3.索引（下标）

- **语法**：`list[索引]`
    - 正向索引：从 `0` 开始
    - 反向索引：从 `-1` 开始

### 4.切片（取出一部分）

- **语法**：`list[start:stop:step]`
    - start：省略默认为 `0`
    - stop：省略默认为 `stop - 1`
    - step：省略默认为 `1`

### 5.修改、添加、删除元素

| 方法                      | 描述                    |
|-------------------------|-----------------------|
| `list[index] = 值`       | 在指定索引修改对应值            |
| `list.append(x)`        | 在末尾添加 x               |
| `list.insert(index, x)` | 在指定索引插入 x             |
| `list.remove(x)`        | 删除指定值 x（如果有多个，只删除第一个） |
| `list.pop()`            | 删除最后一个元素，并返回该元素       |
| `del list[index]`       | 删除指定索引的元素             |

### 6.常用函数

| 函数          | 描述     |
|-------------|--------|
| `len(list)` | 获取列表长度 |
| `sum(list)` | 求和     |
| `max(list)` | 最大值    |
| `min(list)` | 最小值    |

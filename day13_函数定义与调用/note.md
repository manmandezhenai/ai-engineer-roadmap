# Day 13 - 函数定义与调用

## 学到的概念

### 1.函数是什么?

- 函数是一段**有名字的、可重复使用的代码块**
- 内置函数：`print()`、`input()`、`len()`、`type()`

### 2.函数的定义与调用

```python
# 定义
def 函数名(参数1, 参数2, ...):
    函数体
    return 返回值


# 调用
函数名(参数1, 参数2, ...)
```

- `def` 关键字告诉Python："我要定义一个函数"
- 函数名遵循变量命名规则，用小写字母和下划线
- 参数是函数接收的输入变量，可以有多个，也可以没有
- `return` 把结果送回调用处；如果没有 `return`，函数默认返回 `None`

### 3.参数与返回值

- **参数**：函数内部使用的输入变量，调用时传入具体值
- **返回值**：函数执行后的产出，可以用变量接收
- **无返回值的函数**：只负责执行操作（如 `print`），不产出计算值

### 4.作用域

- 函数内部定义的变量叫**局部变量**，只在函数可见
- 函数外部定义的变量叫**全局变量**，整个文件都能访问

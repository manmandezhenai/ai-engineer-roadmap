# Day 7 - 运算符与输入

## 学到的概念

### 1.算术运算符

| 运算符  | 含义        | 示例       | 结果    |
|------|-----------|----------|-------|
| `+`  | 加         | `5 + 3`  | `8`   |
| `-`  | 减         | `5 - 3`  | `2`   |
| `*`  | 乘         | `5 * 3`  | `15`  |
| `/`  | 除（结果为浮点数） | `5 / 2`  | `2.5` |
| `//` | 整除（向下取整）  | `5 // 2` | `2`   |
| `%`  | 取余（模）     | `5 % 2`  | `1`   |
| `**` | 幂运算       | `5 ** 2` | `25`  |

### 2.比较运算符

| 运算符  | 含义   | 示例                 |
|------|------|--------------------|
| `==` | 等于   | `5 == 3` → `False` |
| `!=` | 不等于  | `5 != 3` → `True`  |
| `>`  | 大于   | `5 > 3` → `True`   |
| `<`  | 小于   | `5 < 3` → `False`  |
| `>=` | 大于等于 | `5 >= 5` → `True`  |
| `<=` | 小于等于 | `5 <= 3` → `False` |

### 3.赋值运算符

| 运算符  | 示例                   | 结果    |
|------|----------------------|-------|
| `=`  | x = 10               | `10`  |
| `+=` | x += 5 等同于 x = x + 5 | `15`  |
| `-=` | x -= 3 等同于 x = x - 3 | `12`  |
| `*=` | x *= 2 等同于 x = x * 2 | `24`  |
| `/=` | x /= 4 等同于 x = x / 4 | `6.0` |

#### 4.输入函数

- **概念**：获取用户输入的内容，返回的是**字符串**
- **语法**：`变量名 = input(提示信息)`
- 数字运算前必须 `int()` / `float()` 转换

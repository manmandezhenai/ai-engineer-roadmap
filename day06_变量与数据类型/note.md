# Day 6 - 变量与数据类型

## 学到的概念

### 1.变量

- **概念**：带标签的盒子，用来存储数据
- **语法**：`变量名 = 值`
- **动态类型特性**：Python 是动态类型语言，变量可以修改类型
- 变量命名规则：
    - 只能包含**字母、数字和下划线**，不能以数字开头
    - 不能使用 Python 的关键字（如 `if`、`for`、`class` 等）
    - **严格区分大小写**（`name` 和 `Name` 是两个不同的变量）
    - 建议使用**有意义的英文单词**（如 `user_age` 比 `a` 好得多）

### 2.四种基本数据类型

| 类型         | 说明        | 示例                   |
|------------|-----------|----------------------|
| 整数`int`    | 没有小数点的数字  | `42`、`-7`、`0`        |
| 浮点数`float` | 带小数点的数字   | `3.14`、`-0.5`、`2.0`  |
| 字符串`str`   | 用引号括起来的文本 | `'hello'`、`"python"` |
| 布尔值`bool`  | 只有对/错两种状态 | `True`、`False`       |

- `type()`：可以查看变量的数据类型

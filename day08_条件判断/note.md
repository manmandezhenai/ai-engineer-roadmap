# Day 8 - 条件判断

## 学到的概念

### 1.为什么需要条件判断?

想象一个程序：

- 如果用户是会员，打 8 折；否则原价
- 如果分数 ≥ 60，显示"及格"；否则显示"不及格"

条件判断让程序能**对不同情况做出不同反应**

### 2.基本结构

```python
if 条件1:
    条件1为True时执行的代码块
elif 条件2:
    条件1为False, 条件2为True时执行
else:
    以上所有条件都不满足时执行
```

- `if`：必须要有，一个判断结构的开始
- `elif`：可以有0个或多个，用于检查其他条件
- `else`：可选，处理所有未命中情况

### 3.缩进规则

Python**必须靠缩进**来识别哪几行属于同一个代码块，缩进通过用 `4个空格` / `一个Tab键`

### 4.逻辑运算符

| 运算符      | 描述                            |
|----------|-------------------------------|
| `and`（与） | 两边都为True，才是True               |
| `or`（或）  | 只要一边为True，结果就为True            |
| `not`（非） | 取反（True 变 False、False 变 True） |

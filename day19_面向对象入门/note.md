# Day 19 - 面向对象入门

## 学到的概念

### 1.什么是面向对象?

- **类**（Class）：创建对象的蓝图，描述属性和行为（汽车的设计图纸）
- **对象**（Object）：类的具体实例（具体的汽车）

### 2.定义类

```python
class 类名:
    pass
```

### 3.创建对象并调用方法

```python
对象名 = 类名()

# 访问属性
对象名.属性名

# 调用方法
对象名.函数名()
```

### 4.`__init__` 方法与 `self`参数

- `__init__` 是**初始化方法**，在创建对象时自动运行，用来设定对象的初始属性
- `self` 代表**实例本身**，通过 `self` 可以访问属于该实例的属性和方法
